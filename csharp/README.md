
# Samples for Csharp



## environment

macos: install dotnet CLI:

https://learn.microsoft.com/en-us/nuget/reference/cli-reference/cli-ref-install

add /usr/local/share/dotnet to `PATH`

```bash
brew install mono
brew install nuget
```

project initialized with:

```bash
dotnet new console
```

macos:

```bash
dotnet build
```
